# TechNova Solutions Website

A modern responsive corporate website built using HTML, CSS, Bootstrap 5, and AOS animations.

## Features
- Responsive Navbar
- Bootstrap Carousel
- Services & Team Sections
- Testimonials Slider
- Contact Form
- AOS Animations

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons
- AOS Animation Library

## How to Run
Open `index.html` in any browser.
